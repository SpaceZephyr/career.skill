# 案例：2026 年中，AI 概念股（尤其英伟达）要不要跟

> 用「investing-advisory-board」跑的一次评审会。
> 输入是一个非常常见的问题，看看顾问团如何路由、开会、给条件化结论。

---

## 一、先把问题事实化

在评审前把问题拆成可讨论的事实——

- **场景**：普通个人投资者，手里有闲钱可配置
- **标的**：AI 概念股（以英伟达为代表 + Meta / MS / Google / AMD / 台积电等生态股）
- **时点**：2026 年中，AI 叙事已经持续 3 年多（ChatGPT 2022 底 → 现在 3.5 年）
- **纠结**：错过了 2023-2024 的主升浪，现在跟怕接盘，不跟怕错过下一波

**关键假设**：这是**长期持有**（≥ 3 年）的判断，不是交易性问题。如果假设错了（比如你其实想做波段），结论会变。

---

## 二、路由（谁最相关）

按顾问团分歧地图和路由表：

| 关键问题 | 首选 | 次选 |
|---|---|---|
| 这算不算好生意 / 能不能长期持有 | 段永平 | 《聪明的投资者》 |
| 现在处于周期什么位置 | 霍华德·马克斯 | 《投资最重要的事》 |
| 我是不是在追共识 / 中偏差 | 芒格 | 《穷查理宝典》 |

**—— 这个问题恰好踩「段永平 vs 马克斯」的核心分歧（要不要看周期）**，值得开评审会。

---

## 三、评审会纪要

### 共识（三位专家都同意的）

1. **NVDA 是一门好生意**（护城河来自 CUDA 生态 + 制造和研发规模），这一点没人否认——分歧在**价格**
2. **"AI 会改变世界"是共识**——按第二层思维，共识里没有超额收益，问题是**当前估值有没有把这个共识透支**
3. **不要用杠杆 / 不要用生活费买**——无论怎么看，AI 板块波动率极大，需要"最坏情况下能承受 -70% 且不动"的心态

### 分歧（谁 vs 谁，各自依据的心智模型）

**段永平 vs 马克斯：要不要看周期**

- **段永平**：这是我一辈子的分歧点——**如果 NVDA 是好生意、生意模式我看懂了、10 年后大概率还在赚钱**，那我就买着不动。周期是噪音。"看不懂就不碰"，看懂了就别被波动干扰。**问题落在你自己身上**：你能不能用一段话向外行讲明白 NVDA 靠什么持续赚钱？讲不明白 → 不碰
- **马克斯**：**忽视周期位置的买入是不完整的风险评估**。当前市场的"贪婪信号"打钩数——AI 主题创新叙事、明星科技 CEO 造神、"这次不一样"频繁出现、优质与劣质都在涨、VIX 长期低位——**打钩 ≥ 5 条属于贪婪主导**。这时候进攻不是判断力问题是姿态问题。**不是不能买，是不能重仓 + 不能后期加仓**

**芒格 vs 段永平：判断方式**

- **芒格**：过一遍 25 条误判心理学清单——你是不是因为**社会认同**（大家都在买）+ **剥夺超级反应**（错过了两年不甘心）+ **权威**（明星基金经理背书）+ **承诺一致性**（已经公开看多）**同时命中**？如果 ≥ 3 条，**Lollapalooza 已经发生**，无论理由多充分都要暂停。同时——**你的能力圈到底在哪？** GPU 生态、CUDA 护城河腐蚀速率、Custom ASIC 替代威胁、AI 需求可持续性——**这四件事任何一件你答不上来就是能力圈外**，扔进 too hard pile
- **段永平**：如果你真看懂了、扛得住波动、不用杠杆、是长期钱——**别管别人怎么说、别看短期涨跌，屁股坐得住就行**。用不到的钱十年后再看

### 各自核心建议（每人 ≤ 3 条）

**段永平**
1. 先答"NVDA 靠什么持续赚钱"——答不上来就是不懂，不碰
2. 如果懂，用长期不用的钱买，别看短期。**买了别频繁交易**
3. 别上杠杆、别做空、别被"这次可能真的到顶了"的声音打断

**芒格**
1. 过 25 条误判清单——**特别是社会认同、剥夺反应、Lollapalooza**。三条命中就暂停
2. 用至少 3 门学科看 NVDA：会计（利润率能否维持）+ 竞争动力学（护城河腐蚀）+ 心理学（客户切换意愿）。3 门都通过再买
3. **能力圈外扔进 too hard pile**——芒格自己错过了 Google 一辈子道歉，但没赔过

**马克斯**
1. 打勾当前 9 条"贪婪信号"——**≥ 5 条属于贪婪主导**，你现在的姿态应该是**减仓 / 加防御**而不是加仓
2. **不是不能有 AI 敞口**——但仓位应该是"最坏情况 -70% 你也不改变生活"的规模，通常 ≤ 组合 10-15%
3. **留 ≥ 10% 现金干火药**——钟摆迟早回摆到恐惧那头，那时候是加仓时机

### 方法论检验

**用《聪明的投资者》"防御型选股七条"过 NVDA**：

- [x] 规模足够大 ✅
- [x] 财务状况稳健 ✅
- [x] 长期分红/回购 ⚠️ 分红少但有回购
- [x] 过去 10 年无亏损 ✅
- [x] 10 年 EPS 增长 ≥ 1/3 ✅（远超）
- [ ] **市盈率 ≤ 过去 3 年平均盈利 × 15** ❌ 严重超标
- [ ] **市净率 ≤ 1.5（或 PE × PB ≤ 22.5）** ❌ 严重超标

**未通过 ≥ 2 条 → 不属于防御型标的**。防御型资金按格雷厄姆规则不应买入。

**用《穷查理宝典》Lollapalooza 清单打分**：
- [x] 大家都在买
- [x] 权威人物背书（黄仁勋 / 巴菲特 / 各大机构）
- [ ] 你已经公开表态（因人而异）
- [x] 不买感觉损失巨大（错过 3 年了）
- [x] 好朋友已经赚到钱

→ 4/5 命中，**Lollapalooza 效应存在**。芒格规则：暂停 24 小时。

**用《投资最重要的事》周期温度清单**：
- 融资极其容易 ✅
- IPO 数量创新高 ✅
- "这次不一样" 叙事流行 ✅
- 明星 CEO 被造神 ✅
- VIX 长期极低 ⚠️
- Meme / 主题炒作占主导 ✅

→ **贪婪主导确认**，钟摆已在偏顶部区域。

### 综合结论（条件化）

**如果你是这种情况 → 听段永平**：
- 你从 GPU 时代就深度理解 NVDA 的技术护城河（CUDA 生态 / 芯片架构 / 客户切换成本）
- 用的是 5-10 年不动的闲钱
- 心态过得了"-70% 不改变生活"这一关
- **允许你以合理仓位买（不超过组合 15-20%），买了长期不动**

**如果你是这种情况 → 听马克斯 + 芒格**：
- 是"看别人赚钱怕错过"驱动的
- 讲不清 CUDA 护城河的具体腐蚀速率
- 打算加杠杆或者用生活费的一部分
- 之前没关注过这个行业，最近半年才开始想跟
- **应该等钟摆回摆到中性/恐惧再动手；期间可以先建 3-5% 观察仓熟悉波动，用来教育自己而不是赚钱**

**如果你完全不懂**：
- 扔进 too hard pile
- 用宽基 ETF（含 NVDA 但仓位合理）替代**——芒格明确说过：不懂的人应该买指数**

### 本次评审的盲区

- **具体估值模型 / DCF**：顾问团擅长框架不擅长精算 —— 请转专业算模型的方法论
- **具体拐点判断 / 择时**：马克斯自己反复强调"你无法预测"，不要指望顾问团告诉你"AI 泡沫哪天破"
- **半导体产业链细节**：GPU 架构、CUDA 护城河腐蚀速率、Custom ASIC 威胁 —— 需要专业行业研究员
- **中国 A 股 AI 概念股 / H 股映射**：本顾问团框架偏美股，中国科技股需自行本地化
- **加密 / RWA / 更边缘的 AI 相关标的**：顾问团明确 too hard pile

## 四、我从这次评审里学到的

评审会的价值**不是给我一个答案**，是给我**一张判断地图**：

1. **看到了共识**：三位都同意 NVDA 是好生意 —— 分歧不在这
2. **看到了分歧**：段永平和马克斯在"要不要看周期"上真的有张力，不是废话大和稀泥
3. **看到了偏差**：Lollapalooza 打分 4/5 命中 —— 我在做决策前已经处于**偏差状态**，这本身就是关键信息
4. **看到了自己在哪一边**：把我自己的实际情况套进"综合结论"两栏，我知道我该听谁

**顾问团做不了的**：告诉我"该不该买 NVDA"。这个决定始终是我的。

但它把我原来乱七八糟的焦虑拆成了 4 个具体的判断问题：
- 你懂 NVDA 的护城河吗
- 你打钩了几条 Lollapalooza
- 你的钱是几年不动的
- 你能不能扛 -70%

**答完这 4 个，答案就出来了。**
